# 향수 검색기

관리자 비밀번호를 이용해 향수를 추가할 수도 있고, 조건에 따라 향수를 검색할 수도 있는 파이썬 프로젝트입니다.
tkinter 라이브러리를 이용하여 제작했습니다.
